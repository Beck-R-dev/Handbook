Комментарии

Комментарии — это поясняющий текст в коде, который компилятор игнорирует, и предназначен этот текст для разработчиков. Комментарии бывают двух видов: однострочные и многострочные. Разберём сначала однострочные. Чтобы написать такой комментарий, нужно поставить две косые черты (//), а после — сам комментарий. Например:

internal class Program {
static void Main(string[] args) {
// Это однострочный комментарий!
}

Всё, что идёт в строке после двух косых черт, будет считаться комментарием, даже если это будет часть кода:

В данном примере Console.WriteLine("Text"); считается комментарием, а не командой, поэтому компилятор будет его игнорировать. В этом можно убедиться, если скомпилировать и запустить программу:


*path*\ConsoleApp1\ConsoleApp1\bin\Debug\CondoleApp1.exe (process *number*) exited with code 0 (0x0).
Press any key to close this window . . .

Как видите, в консоль не вывелась надпись "Text".

Обращаю внимание,

что пример выше, это демонстрация того, что комментарии могут быть в любом месте кода, но в реальных программах комментарии не пишутся просто так — они должны что-то пояснять в том участке кода, в котором используются.

Т.е. всё работает корректно т.к. синтаксис языка не нарушен. Однако на практике такое использование комментариев встречается крайне редко.

В большинстве редакторов (IDE) есть горячие клавиши, которые позволяют закомментировать одну или более строк. В Visual Studio это:

  • Ctrl + / — преобразует в однострочный комментарий строку, на которой находится каретка, или все строки, которые выделены хотя бы частично;
  • Ctrl + Shift + / — преобразует в многостроынй комментарий именно то место, где находтся каретка, или выделенный участок.

Типы данных и переменные

Язык программирования — это инструмент, позволяющий манипулировать данными, и, для того, чтобы мы смогли это сделать нам нужно знать 3 вещи:

  • Зачем нужен тип данных
  • Что такое переменная
  • Что такое присваивание

Тип данных указывает компилятору, как хранить данные в памяти. Дело в том, что компьютер хранит разные данные по-разному: например, целые числа (1, 3, 15, -5 и т.п.) он хранит одним образом, а дробные числа (1.5, -3.2, 3.33 и т.п.) — другим. Поэтому важно указывать компилятору, какой именно тип данных мы собираемся использовать.

Обращаю внимание,

Что в данном блоке мы не будем рассматривать, как именно хранятся те или иные данные в памяти, т.к. на данном этапе это будет излишне.

Существует много типов данных, но в данном блоке мы разберём 4:

  • int — целые числа;
  • float — вещественные числа (дробные);
  • double — тоже вещественные, но более точные;
  • string — строки.

Переменная — это имя, с помощью которого мы обращаться к данным в коде. Имя может состоять из латинских букв (a-z, A-Z), цифр (0-9) и символа подчёркивания(_), но не может начинаться с цифры. Также регистр символов имеет значение: например, имена "Number" и "number" считаются разными переменными, так как первая буква отличается по регистру.

Присваивание — это операция, которая "прикрепляет" данные к переменной, чтобы мы могли потом обращаться к ней. Для присваивания используется символ =, например, чтобы присвоить переменной "Number" число 2, нужно написать Number = 2; (двойка является числовым данным).

А теперь научимся объявлять переменную. Для этого нужно написать тип данных, затем, через пробел, имя переменной: int myVariable;
