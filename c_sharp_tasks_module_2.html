Задачи на переменные

1. Напишите программу, которая выводит на консоль письмо, содержащее данные об имени, возрасте и названии города, которые будут являться переменными.

Пример:

Здравствуй Тимур!

Хочу поздравить тебя с днём рождения! Тебе уже 28!
Как там город Ташкент? Как погода? Надеюсь всё хорошо!
Обязательно расскажи, что у тебя нового! С нетерпением жду твоего ответа в ближайшее время.

С уважением, Игорь.

*path*\ConsoleApp1\ConsoleApp1\bin\Debug\CondoleApp1.exe (process *number*) exited with code 0 (0x0).
Press any key to close this window . . .

Небходимые знания:

  • Console.Write() / Console.WriteLine()
  • Управляющий символ переноса строк;
  • Переменные.

2. Напишите программу, которая конвертирует доллары в суммы

Необходимые знания:

  • Курс. Например, $1 = 12500 сум
  • Console.Write() / Console.WriteLine()
  • Переменные;
  • Арифметические ператоры;
  • Преобразование типов данных.

3. Напишите программу, которая конвертирует значение градусов Цельсия в значение градусов Фаренгейта

Необходимые знания:

  • Формула перевода из Цельсия в Фаренгейт: F = C * 1.8 + 32, где F — Фаренгейты, а C — Цельсии;
  • Console.Write() / Console.WriteLine()
  • Переменные;
  • Арифметические ператоры;
  • Преобразование типов данных.
