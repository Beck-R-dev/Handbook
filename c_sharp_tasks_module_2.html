Задачи на переменные и операторы

1. Выведите на консоль текст письма, содержащей данные об имени, возрасте и названии города, которые будут являться переменными.

Пример:

Здравствуй Тимур!

Хочу поздравить тебя с днём рождения! Тебе уже 28!
Как там город Ташкент? Как погода? Надеюсь всё хорошо!
Обязательно расскажи, что у тебя нового! С нетерпением жду твоего ответа в ближайшее время.

С уважением, Игорь.

*path*\ConsoleApp1\ConsoleApp1\bin\Debug\CondoleApp1.exe (process *number*) exited with code 0 (0x0).
Press any key to close this window . . .

Небходимые знания:

  • Console.Write() / Console.WriteLine()
  • Управляющий символ переноса строк;
  • Переменные.

2. Выведите на консоль сумму двух введёных пользователем чисел

Необходимые знания:

  • Console.Write() / Console.WriteLine()
  • Переменные;
  • Арифметические операторы;
  • Преобразование типов данных.

3. Конвертируйте доллары в узбекские суммы и выведите результат на консоль. Значение долларов вводит пользователь

Необходимые знания:

  • Курс. Например, $1 = 11950 сум
  • Console.Write() / Console.WriteLine()
  • Переменные;
  • Арифметические операторы;
  • Преобразование типов данных.

Enter the number: 123
